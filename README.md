# AI × Redesign — 허강범 포트폴리오

기존에 직접 진행한 4개 브랜드 리디자인을 **Claude Code와 협업**해 UX·인터랙션·코드 품질까지
한 단계 더 발전시킨 프로젝트. AI가 대신 만든 것이 아니라, 사람이 주도하고 AI와 함께 발전시킨 기록입니다.

## 구성
- `index.html` — 포트폴리오 메인 (가로 갤러리)
- `works/ai-redesign.html` — **AI × Redesign 종합 소개 허브** (4개 브랜드 한눈에)
- `works/<brand>/` — 브랜드별 라이브 리디자인(v2) + `casestudy-ai.html`(Before↔After 비교 케이스 스터디)
  - Hasselblad · Yamaha · Maruni · JW중외제약

## 만든 것
- 여정 완결(제품/가구 상세 딥다이브 + View Transitions + 종착 문의 섹션)
- 브랜드별 시그니처 연출(WebGL 빛 · 사운드 링 맥동 · material 줌 · 데이터 파티클)
- 실서비스 수준 코드 품질(60fps · 접근성 · reduced-motion · 반응형)

Stack: HTML · CSS · JavaScript · GSAP · Three.js/OGL · View Transitions · Lenis
